Point of Sales Download Unsaved Orders Odoo Apps

Backup & Restore POS - Point of Sale Odoo App allows users to safeguard and recover their unsaved Point of Sale (POS) orders with ease. Users can effortlessly back up these orders in JSON format, making it convenient to restore them later. Additionally, users can create new POS orders directly from the restored data. In addition, they can also delete unsaved POS orders without the need to download them first. This odoo app improves the process of managing and recovering POS orders, enhancing the efficiency of businesses using Odoo.

Features

Backup/Restored Unsaved POS Order

User can easily backup/restored the unsaved POS order.

Download Unsaved POS Orders

User have option to download the unsaved POS orders in JSON format.

Import JSON Order Data

User can import the JSON order data and create the POS order directly from restored orders data.

Delete Unsaved POS Orders

User have option to delete the unsaved POS orders without downloading it.

Added Product into Cart

Download Unsaved POS Orders

When user click on Backup/Restore icon, One wizard would be open in that having two option 'Download Unsaved Orders' and 'Delete Unsaved Orders'.

Confirmation Popup

Once downloaded unsaved orders, Confirmation popup should be open for remove locally stored unsaved order.

User can see the locally stored unsaved order should be deleted.

JSON Order Data Import Menu

Goto Point of Sale -> Order Restore/Backup -> JSON Order Data Import menu to import downloaded unsaved order data in JSON format.

Import Data Wizard

User have to upload the JSON file in this wizard and also they have option to 'IMPORT DATA' for import only data and 'IMPORT DATA + CREATE ORDER' for import data with created POS order.

Restored Orders Data Menu

Goto Point of Sale -> Order Restore/Backup -> Restored Orders Data menu to view the restored imported orders data.

Create POS Order from Restored Orders Data

User can create POS order from restored orders data by clicking on 'CREATE ORDER' button.

Created POS Order from Restored Orders Data

User can view the created POS order directly from restored orders data.

Import Data Wizard

User have to upload the JSON file in this wizard and clicking on 'IMPORT DATA + CREATE ORDER' option to import data with created POS order.

Created POS Order from Restored Orders Data

User can view the created POS order directly from imported restored orders data.

Delete Unsaved POS Orders

User can directly delete unsaved POS order by clicking on 'Delete Unsaved Orders' option.

Confirmation Popup

Confirmation popup should be open for remove locally stored unsaved order.

User can see the locally stored unsaved order should be deleted.

Warning/Validation

If user tries to importing existing orders data, Warning raised.
